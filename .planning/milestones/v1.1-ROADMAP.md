## Milestone Goal

Make Git-based version control accessible to non-technical Alteryx analysts via a Windows desktop companion app (local web server, system tray, auto-start) and polished CI integration.

## Key Accomplishments

1. **Windows desktop companion app** — PyInstaller `.exe` bundles FastAPI + React + acd CLI; auto-launches browser, runs in system tray, starts on Windows boot. Zero Python install required.
2. **Core version control loop** — File watcher (native + polling for SMB/UNC), Save Version (selective commit with message), History timeline, ACD diff viewer embedded inline, Undo last save, Discard to `.acd-backup`.
3. **Remote backup** — GitHub OAuth device flow + GitLab PAT; credentials in OS keyring; single-button push with auto-create remote repo; ahead/behind indicator; in-app PR/MR creation from experiment branches.
4. **Branch management** — Experiment copies (`experiment/YYYY-MM-DD-*`), branch switch, git graph view with multi-branch SVG, per-commit push status cloud icons.
5. **CI integration** — GitHub Actions comment deduplication (find-or-update), inline PNG graph, GitLab CI marker-based MR comment dedup, `ci-templates/` distributable with setup README.
6. **HTML report redesign** — Dark-first CSS variable theming, stat cards with accent colors, 3px left-accent section headers, animated chevron tool rows, Before/After diff panels, `localStorage` theme persistence — all self-contained, zero CDN.
