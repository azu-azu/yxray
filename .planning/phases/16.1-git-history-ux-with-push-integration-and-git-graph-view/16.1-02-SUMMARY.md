## Accomplishments
- Extended CommitEntry interface with `is_pushed: boolean` field and added RemoteStatus type to HistoryPanel
- Cloud icon (lucide-react Cloud) renders on each history entry with tooltip showing connected remotes when remoteConnected and entry.is_pushed; hidden entirely when no remote connected
- Push button "Back up N versions" appears in HistoryPanel header when ahead > 0 and a remote is connected; calls /api/remote/push for all connected providers in parallel
- RemotePanel now accepts and calls `onPushComplete` after each successful push
- AppShell wires push completion signals: `setLastPushTimestamp(Date.now())` triggers HistoryPanel remote status re-fetch; `fetchHistory()` updates is_pushed flags on all entries
