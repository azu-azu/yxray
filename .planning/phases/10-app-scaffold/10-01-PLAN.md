    Create app/server.py:
    - Import FastAPI, StaticFiles, sys, Path
    - Define _static_dir() function: if getattr(sys, "frozen", False) use Path(sys._MEIPASS) / "frontend" / "dist", else use Path(__file__).parent / "frontend" / "dist"
    - Create FastAPI() instance as `app`
    - Define @app.get("/health") returning {"status": "ok", "version": version("alteryx-diff")} using importlib.metadata.version
    - Mount StaticFiles LAST: app.mount("/", StaticFiles(directory=str(_static_dir()), html=True), name="frontend") — wrapped in try/except so missing dist/ dir doesn't crash the server during dev/test (log a warning instead)

    Create app/main.py:
    - import multiprocessing, socket, asyncio, webbrowser, threading, uvicorn
    - from app.server import app as fastapi_app
    - find_available_port(start=7433, count=11): loop range(start, start+count), attempt sock.bind(("127.0.0.1", port)), return (port, sock) on first success, raise OSError if all fail
    - sock.setsockopt(socket.SOL_SOCKET, socket.SO_REUSEADDR, 1) before bind
    - main(): call find_available_port(); print startup log with actual port; schedule webbrowser.open via threading.Timer(1.0, lambda: webbrowser.open(f"http://localhost:{port}")); build uvicorn.Config(fastapi_app, host="127.0.0.1", port=port, log_level="warning"); run asyncio.run(server.serve(sockets=[sock]))
    - CRITICAL — the __main__ guard must be structured exactly as follows (freeze_support BEFORE main()):

      ```python
          multiprocessing.freeze_support()  # MUST be first — prevents infinite spawn on Windows onefile
          main()
      ```
      Do NOT call freeze_support() inside main(). It must execute before main() is called so Windows process spawn protection is active before any uvicorn worker processes are created.
