  <behavior>
    _compute_state(status_data: dict) → (str, str):
    - total_changes = sum(v.get("changed_count", 0) for v in status_data.values())
    - any_watching = bool(status_data)
    - if total_changes > 0: return ("changes", f"Alteryx Git Companion — {total_changes} change{'s' if total_changes != 1 else ''} detected")
    - elif any_watching: return ("watching", "Alteryx Git Companion — watching")
    - else: return ("idle", "Alteryx Git Companion")

    TrayIcon class:
    - __init__(self, port: int, server: uvicorn.Server): store port and server ref; load 3 PIL Images
    - _load_image(name: str) → PIL.Image.Image: _get_asset_path(name) using sys._MEIPASS pattern from server.py; fallback to programmatic generation
    - _get_asset_path(filename: str) → Path: if frozen → Path(sys._MEIPASS) / "assets" / filename; else → Path(__file__).parent.parent / "assets" / filename
    - Programmatic icon generation (fallback and for placeholder files):
      - idle: existing assets/icon.ico if present; else 64x64 white square
      - watching: 64x64 green (#22c55e) square with 'W' letter
      - changes: 64x64 amber (#f59e0b) square with '!' letter
    - run(self): build pystray.Icon with Menu; call icon.run() with setup=self._setup (blocking in daemon thread)
    - _setup(self, icon): starts polling loop in a threading.Thread(daemon=True)
    - _poll_loop(self, icon): while icon.visible: requests.get status, compute state, update icon.icon and icon.title; time.sleep(5)
    - Tray menu: MenuItem("Open Alteryx Git Companion", on_open, default=True) + MenuItem("Quit", on_quit)
    - on_open: webbrowser.open(f"http://localhost:{self.port}")
    - on_quit: icon.stop() then self.server.should_exit = True

    start_tray(port: int, server: uvicorn.Server) → None:
    - t = TrayIcon(port, server); t.run()
    - Called from daemon thread by main()
