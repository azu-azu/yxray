Backend API responses:
- GET  /api/remote/status?project_id=X&folder=Y&provider=github
  → {ahead: number, behind: number, github_connected: boolean, gitlab_connected: boolean, repo_url: string | null}
- POST /api/remote/github/start
  → {user_code: string, verification_uri: string}
- GET  /api/remote/github/status
  → {connected: boolean}
- POST /api/remote/github/connect body: {token: string}
  → {connected: boolean}   ← GitHub PAT fallback endpoint (stores via SERVICE_GITHUB keyring)
- POST /api/remote/gitlab/connect body: {token: string}
  → {connected: boolean, error?: string}
- POST /api/remote/push body: {project_id: string, folder: string, provider: string}
  → {success: boolean, repo_url?: string, created?: boolean, error?: string}
