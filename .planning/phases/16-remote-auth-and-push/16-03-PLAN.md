GIT_ASKPASS pattern (from RESEARCH.md):
```python
# Write temp askpass script, set GIT_ASKPASS env var, delete after push
with tempfile.NamedTemporaryFile(mode="w", suffix=".py", delete=False, prefix="acd_askpass_") as f:
    f.write(f"#!/usr/bin/env python3\nimport sys\nprint('{token}')\n")
    askpass_path = f.name
os.chmod(askpass_path, stat.S_IRWXU)
try:
    env = os.environ.copy()
    env["GIT_ASKPASS"] = askpass_path
    env["GIT_TERMINAL_PROMPT"] = "0"
    subprocess.run(["git", "-C", folder, "remote", "set-url", "origin", remote_url], env=env, ...)
    subprocess.run(["git", "-C", folder, "push", "-u", "origin", "HEAD"], env=env, check=True, ...)
finally:
    os.unlink(askpass_path)
```

git_ahead_behind pattern:
```python
# Step 1: check if upstream exists
r = subprocess.run(["git", "-C", folder, "rev-parse", "--abbrev-ref", "--symbolic-full-name", "@{u}"], ...)
if r.returncode != 0: return (0, 0)
tracking = r.stdout.strip()  # "origin/main"
# Step 2: count ahead/behind
ahead = subprocess.run(["git", "-C", folder, "rev-list", "--count", f"{tracking}..HEAD"], ...)
behind = subprocess.run(["git", "-C", folder, "rev-list", "--count", f"HEAD..{tracking}"], ...)
return (int(ahead.stdout.strip() or "0"), int(behind.stdout.strip() or "0"))
```

Router design (all endpoints):
- POST /api/remote/github/start — async def; calls remote_auth.request_device_code(); fires asyncio.create_task(remote_auth.poll_and_store(...)); returns {user_code, verification_uri}
- GET  /api/remote/github/status — def (sync); calls remote_auth.get_github_token(); returns {connected: bool}
- POST /api/remote/github/connect — def (sync); body {token: str}; calls remote_auth.store_github_token(token); returns {connected: bool}  ← GitHub PAT fallback endpoint
- POST /api/remote/gitlab/connect — def (sync); body {token: str}; validates + stores; returns {connected: bool, error?: str}
- GET  /api/remote/gitlab/status — def (sync); returns {connected: bool}
- POST /api/remote/push — def (sync, runs in threadpool); body {project_id, folder, provider}; gets token from keyring; gets remote_url from config_store; if no remote_url, calls auto-create then saves URL; calls git_push; returns {success: bool, repo_url?: str, created?: bool}
- GET  /api/remote/status — def (sync); body requires project_id + folder query params; calls git_fetch + git_ahead_behind; returns {ahead, behind, github_connected, gitlab_connected, repo_url}
