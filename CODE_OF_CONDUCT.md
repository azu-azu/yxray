# Code of Conduct

This project adopts the [Contributor Covenant](https://www.contributor-covenant.org/version/2/1/code_of_conduct/) version 2.1 as its code of conduct.

All participants are expected to uphold this code. Please report unacceptable behavior via [GitHub Security Advisories](https://github.com/Laxmi884/alteryx-git-companion/security/advisories/new) or by opening a private GitHub issue.

For the full text, see: https://www.contributor-covenant.org/version/2/1/code_of_conduct/
